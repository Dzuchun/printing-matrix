Module type_matrux::object

source ·
Expand description

Contains data objects as well as some infrastructure for their definition and usage.

All of the provided objects are purposely immutable - this way you can guarantee that actual data stored inside them was indeed returned from a site. For validation purposes, they all have [get_age] function, returning [time::Duration] representing time passed since their creation.
